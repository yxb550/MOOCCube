Dataset Information
The core dataset used in the experiments of this project, MOOCCube, is publicly available and can be accessed via the following official website:


Access Link:  http://moocdata.cn/data/MOOCCube

Dataset Description

MOOCCube is a large-scale, real-world Massive Open Online Course (MOOC) dataset released by Tsinghua University and the XuetangX platform. It provides a comprehensive, course-centered knowledge graph (KG) structure that incorporates rich pedagogical and semantic relationships.
